# Image Data Loader
In this project a data loader was created to load image data from different folders

## Tools
- Python
- NumPy
- Matplotlib
- OpenCV
- TensorFlow

## About Data
There are images of 6 different classes. Each class contains 30 images seperated in individual folders.
